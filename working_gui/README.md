# Working GUI Directory

This directory contains the main working GUI implementations.

## Files:
- `crash_proof_enhanced_gui.py` - **RECOMMENDED** - Guaranteed no crashes when clicking tabs
- `optimized_enhanced_gui.py` - Performance optimized version with timeouts and retries
- `standalone_enhanced_gui.py` - No external dependencies version

## Usage:
Use `crash_proof_enhanced_gui.py` for the most stable experience.
All versions include the full feature set with different optimization approaches.

## Features:
- ✅ Lazy tab loading
- ✅ Timeout protection  
- ✅ Error handling
- ✅ Memory management
- ✅ Resource monitoring
- ✅ VoxSigil styling

## Launching:
Use the main launcher in the root directory: `Launch_VoxSigil_Main.bat`
